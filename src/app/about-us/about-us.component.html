About the creators

This alternative version of the Google Chrome Dinosaur game was created by UCI Student

Leon Vay for the Fall 2021 Informatics 133 Class.

Leon is a 4th year at UCI pursuing the Informatic Major.

About the Game

The game was created using Angular with the combination of the Tensorflow

handtracker Javascript File. The original game by Google was a very simple

design and the creators of this web app decided to follow in its footsteps

and keep the design simple as well.

Going back to the game

To go back to the game, hit the start button at the bottom then make this gesture

  • Navigate to Game - One Open Hand One Closed Hand
